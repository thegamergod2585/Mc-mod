# Minecraft Bedrock X-ray Resource Pack

This is a minimal resource pack structure. The main effect comes from modified block textures.

## Folder Structure

```
resource_pack/
├── manifest.json
├── textures/
│   └── blocks/
│       ├── stone.png
│       ├── deepslate.png
│       ├── diamond_ore.png
│       ├── gold_ore.png
│       ├── iron_ore.png
│       ├── copper_ore.png
│       ├── emerald_ore.png
│       ├── redstone_ore.png
│       ├── lapis_ore.png
│       └── coal_ore.png
└── texts/
    └── en_US.lang
```

## To Customize

1. Replace PNG textures in `textures/blocks/` with your own
2. Make stone semi-transparent by reducing opacity in your image editor
3. Make ores brighter with more vibrant colors
4. Re-add the pack to Minecraft to see changes
